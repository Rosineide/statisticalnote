{{% notice note %}} Add the line relativeurls = true to the config.toml file. This is required. Otherwise, the links to css files will be broken when the site is hosted on github. {{% /notice %}}

theme = "hugo-theme-learn"
baseURL = "http://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
relativeurls = true

Adding the following lines to the config.toml file will make the site searchable.

[outputs]
home = [ "HTML", "RSS", "JSON"]

Change site colors

[params]
  # Change default color scheme with a variant one. Can be "red", "blue", "green".
  themeVariant = "blue"
